\section{Artifact Checklist}

Final submission should include source code, tests, scripts, README, AI disclosure, this report, and selected evidence artifacts. Simulator artifacts under \code{reports/evidence/simulation} should include request snapshots, execution logs, execution summaries, child-order CSVs, fill CSVs, timelines, and TWAP ledgers. The included raw Testnet rejection artifacts under \code{reports/evidence/testnet} additionally include \code{symbol\_rules.json}, \code{reconciliation\_orders.csv}, and \code{evidence\_manifest.json}. Accepted Binance Testnet artifacts, when available after account funding or margin configuration, should show non-empty exchange order IDs and accepted exchange-order evidence.

Artifacts must not include API secrets, listen keys, request signatures, signed payloads, or private account data not needed for review.
