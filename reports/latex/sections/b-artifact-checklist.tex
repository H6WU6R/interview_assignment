\section{Artifact Checklist}

Final submission should include source code, tests, scripts, README, AI disclosure, this report, and selected evidence artifacts. Simulator artifacts under \code{reports/evidence/simulation} should include request snapshots, execution logs, execution summaries, child-order CSVs, fill CSVs, timelines, and TWAP ledgers. The accepted sanitized Testnet artifacts under \code{reports/evidence/testnet/chase/exec\_3168600ee25b4193} and \code{reports/evidence/testnet/twap/exec\_85bef3985ea3431a} additionally include \code{symbol\_rules.json}, \code{reconciliation\_orders.csv}, and \code{evidence\_manifest.json} with non-empty exchange order IDs and accepted exchange-order evidence. The older insufficient-margin artifacts remain only rejected connectivity/error evidence.

Artifacts must not include API secrets, listen keys, request signatures, signed payloads, or private account data not needed for review. Accepted Testnet logs redact \code{ACCOUNT\_UPDATE} balance and position fields while preserving order/trade updates needed to prove exchange acceptance.
